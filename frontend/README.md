# AI-Ops 前端界面

这是AI应用运维助手的前端界面，基于React构建，提供用户友好的交互体验。

## 功能特性

- 问题描述输入界面
- AI解决方案展示界面
- 加载状态指示器
- 响应式设计

## 技术栈

- React.js
- Tailwind CSS
- Fetch API

## 组件说明

### App.js

主应用组件，负责整体布局和状态管理。

### ProblemInput.js

问题输入组件，包含：
- 文本域用于输入问题描述
- 提交按钮触发AI解决方案生成
- 加载状态处理

### SolutionDisplay.js

解决方案展示组件，包含：
- 加载动画
- 解决方案内容展示
- 空状态提示

## 安装与运行

1. 安装依赖：
   ```
   npm install
   ```

2. 启动开发服务器：
   ```
   npm start
   ```

   默认情况下，前端应用将在 http://localhost:3000 运行。

## 构建生产版本

```
npm run build
```

构建后的静态文件将位于 `build/` 目录中，可以部署到任何静态文件服务器上。
